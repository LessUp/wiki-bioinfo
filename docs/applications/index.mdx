# 分析方向与案例

这一部分承接前面的基础与方法层，把它们重新放回具体分析任务中：DNA-seq、RNA-seq、宏基因组和系统发育等问题，最终都需要通过工作流和案例来理解。

## 这一部分在全站中的位置

它是整站的应用层，用来回答“前面学的那些方法最后到底用在什么地方”。

## 为什么重要

如果只有方法页，没有应用层，读者很容易形成“知道术语，但不知道任务边界”的知识碎片化问题。

## 推荐阅读顺序

1. [工作流与案例](../workflows/index.md)
2. [变异检测](../variants/index.mdx)
3. [转录组](../transcriptomics/index.mdx)
4. [系统发育与进化](../phylogeny/index.md)
