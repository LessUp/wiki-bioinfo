# 组装与图算法

组装问题关注如何从大量短片段或长读段重建原始基因组或转录本序列。

这是从“局部观测恢复整体结构”的经典计算问题。

## 这一部分在全站中的位置

这一节连接 reads、coverage、错误模型与图算法，重点回答：为什么局部读段信息会在图里形成路径问题，以及这些路径为什么难解。
