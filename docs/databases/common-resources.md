# 常用数据库与资源

## 核心数据库

- **NCBI**：综合入口，覆盖序列、文献、注释等多类资源；
- **Ensembl**：基因组浏览与注释；
- **UniProt**：蛋白质序列与功能信息；
- **PDB**：蛋白质与核酸三维结构；
- **KEGG**：通路与功能网络；
- **GEO / SRA**：公开表达与测序数据。

## 为什么要单独学习数据库

真实分析工作的很多时间都花在：

- 找正确的数据；
- 确认参考版本；
- 理解字段和注释含义；
- 把多个资源对应起来。

## 建议

遇到一个分析问题时，尽量先明确：

1. 需要哪个参考基因组版本；
2. 需要哪类注释文件；
3. 结果要与哪个数据库对象对应。
