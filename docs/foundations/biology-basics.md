# 生物信息学中的基础对象

## 是什么

生物信息学研究的核心对象通常包括：

- DNA 序列
- RNA 序列
- 蛋白质序列
- 基因组与转录组
- 测序读段（reads）
- 注释信息（annotation）

这些对象最终都会被表示成计算系统能够处理的数据结构。

## 为什么重要

如果不先理解这些对象，就很难明白：

- 为什么比对问题可以看成字符串匹配；
- 为什么组装问题会转化成图问题；
- 为什么表达量分析依赖注释文件和坐标系统；
- 为什么错误模型、覆盖度和重复序列会影响算法结果。

## 基本对象之间的关系

一个常见的数据流是：

1. 生物样本被测序，产生 reads；
2. reads 与参考基因组进行比对，或直接做 de novo 组装；
3. 比对或组装结果再结合注释文件进行解释；
4. 最终输出表达量、变异、结构特征或进化关系。

## 应用场景

- WGS/WES 变异检测
- RNA-seq 表达分析
- 宏基因组组装与分类
- 蛋白质序列功能预测
