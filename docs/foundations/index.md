# 基础与数学

生物信息学的很多问题都可以抽象成对序列、图、矩阵和概率模型的计算。

但在进入算法之前，先要明确几个基本对象：

- 序列是什么；
- 读段和参考基因组是什么；
- coverage、错误模型和坐标系统怎样影响后续分析；
- 注释和特征区间如何把序列映射到生物学对象；
- 为什么图模型、动态规划和概率模型会反复出现。
