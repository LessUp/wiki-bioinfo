# 序列、字符串与坐标系统

## 是什么

在计算机视角下，DNA、RNA 和蛋白质序列都可以看作有限字母表上的字符串：

- DNA：`{A, C, G, T}`
- RNA：`{A, C, G, U}`
- 蛋白质：20 个氨基酸字符

这使得许多生物信息学问题可以借助字符串算法来处理。

## 为什么重要

一旦把序列抽象成字符串，就可以自然引入：

- 子串与模式匹配
- 编辑距离
- 动态规划
- 索引结构
- 压缩表示

## 坐标系统

除了序列本身，还要理解区间坐标：

- 起点、终点如何定义；
- 不同文件格式可能采用不同的闭区间或半开区间约定；
- 正负链方向会影响对特征的解释。

这些看似细节的问题，在注释、可视化和流程串联时经常引起错误。

## 例子

如果一个读段在参考基因组上定位到某个外显子区间，那么它的意义依赖于：

- 使用的参考版本；
- 坐标系统约定；
- 对链方向和剪接结构的理解。
