# BioInfo Wiki 简介

BioInfo Wiki 是一个面向中文社区的开源生物信息学知识库。

它参考 OI Wiki 的组织方式：

- 用清晰的层级结构串起知识点
- 用统一的文档模板降低阅读和贡献门槛
- 用 GitHub 协作机制持续积累内容

但它的主题聚焦于 **生物信息学**，目标是把以下几类知识连接起来：

1. 生物学对象：DNA、RNA、蛋白质、基因组、转录组。
2. 算法模型：字符串、图、动态规划、概率模型、索引结构。
3. 工具与流程：比对、组装、注释、定量、变异检测、数据库检索。

## 这个项目想解决什么问题

生物信息学的资料很多，但常见问题是：

- 生物学背景和算法背景常常分散在不同体系里；
- 新手能找到工具教程，却难理解工具背后的原理；
- 算法教材强调模型和推导，和实际流程之间的桥梁不够明显；
- 很多中文资料零散、过时，缺少持续维护机制。

BioInfo Wiki 的第一阶段会优先建设一套可扩展的站点骨架和文档规范，再围绕经典教材主线逐步扩充内容。

## 第一阶段重点

- 建立站点骨架与统一导航
- 搭建生物信息学核心知识目录
- 补齐项目介绍、学习路线、贡献说明、写作规范
- 编写首批围绕经典算法与核心概念的中文页面
