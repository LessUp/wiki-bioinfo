# 连锁不平衡

## 是什么

连锁不平衡（Linkage Disequilibrium, LD）描述的是不同位点上的等位基因并非独立组合，而是以高于或低于随机期望的频率共同出现。

如果两个 SNP 完全独立，那么知道一个位点的等位基因并不能帮助预测另一个位点；而在高 LD 区域，一个 SNP 往往能部分代表附近多个 SNP。

## 为什么会有 LD

- 两个位点在物理距离上接近，重组不容易把它们打散；
- 群体历史事件（瓶颈、扩张、混合）改变了单倍型结构；
- 选择会把某些组合一起保留下来。

## 常见指标

### D 与 D'

`D` 衡量观察到的单倍型频率与独立假设下频率的差：

$$D = P_{AB} - p_A p_B$$

但 `D` 受等位基因频率影响大，因此常用标准化的 `D'`。

### r²

GWAS 和 tag SNP 场景里最常用的是：

$$r^2 = \frac{D^2}{p_A(1-p_A)p_B(1-p_B)}$$

`r²` 越接近 1，说明两个位点的预测能力越强。

## haplotype block

在真实基因组中，LD 不是均匀分布的，而是形成一个个 block：

- block 内重组少，LD 高；
- block 之间重组多，LD 低。

这也是为什么很多芯片只需测少量 tag SNP，就能覆盖附近的大量变异信息。

## 在生物信息学中的用途

- **GWAS**：显著位点附近常有一串高 LD SNP，关联信号不一定来自最显著的那个 SNP 本身；
- **Fine-mapping**：利用 LD 缩小潜在因果变异集合；
- **Imputation**：借助参考群体的 haplotype 结构补全未测位点；
- **群体历史推断**：LD 衰减速度与有效群体大小、重组率有关。

## 可视化

最常见的是 LD heatmap：

- 横纵轴都是 SNP 顺序；
- 方格颜色表示 `r²` 或 `D'`；
- 高 LD block 常表现为对角线上的深色三角结构。

## 常见误区

- **显著 SNP 就是因果 SNP**：很多时候只是和真正因果位点处在高 LD；
- **LD 是固定不变的**：不同祖源群体的 LD 结构差异很大；
- **距离近就一定高 LD**：虽然常见，但重组热点会打破这种关系。
