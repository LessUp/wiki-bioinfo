# 转录组

<PageHeaderMeta
  section="Applications"
  audience="已经接触 RNA-seq 或想理解表达定量、isoform 归属与统计建模的读者"
  startWith="先理解序列索引、比对和 RNA-seq 工作流，再读 pseudo-alignment"
/>

转录组分析关注的是：给定 RNA-seq 等数据，如何理解基因和转录本的表达、剪接、条件差异以及这些变化背后的生物学含义。

## 这一部分在全站中的位置

这一节位于“分析方向与案例”之下，承接索引、比对和概率模型，把它们重新放回 RNA-seq 的定量与表达解释问题中。

## 为什么这一节重要

RNA-seq 看起来像是“把 reads 比到转录组上再数一数”，但真正困难的地方在于：

- 一个 read 可能兼容多个 isoform；
- 基因层与转录本层的问题并不相同；
- 定量和差异分析都依赖统计假设；
- 注释版本变化会直接改变结果解释。
