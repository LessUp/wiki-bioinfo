## 1. 核心数学模型

仿射 Gap 代价函数定义为：
$$g(k) = \rho + k \cdot \sigma$$
其中：
- **$\rho$ (Gap Opening Penalty)**：开启一个新 Gap 的固定代价。
- **$\sigma$ (Gap Extension Penalty)**：每增加一个碱基的额外代价。
- **$k$**：Gap 的长度。

**直觉**：由于 $\rho \gg \sigma$，算法会倾向于让 Gap 尽量连续，而不是在序列中到处乱开小坑。

## 2. 状态转移：三矩阵模型

引入仿射罚分后，简单的二维 DP 矩阵已无法区分“当前是否处于 Gap 中”。我们需要维护三个平行的矩阵：

  items={[

    },
    {
      term: '矩阵 $I_{i,j}$',
      definition: '表示比对以 $y_j$ 处的插入（对 $x$ 而言是 Gap）结尾时的最高得分。',
    },
    {
      term: '矩阵 $D_{i,j}$',
      definition: '表示比对以 $x_i$ 处的删除（对 $y$ 而言是 Gap）结尾时的最高得分。',
    },
  ]}
/>

### 递推公式
$$
\begin{aligned}
  M_{i,j} &= s(x_i, y_j) + \max \{ M_{i-1,j-1}, I_{i-1,j-1}, D_{i-1,j-1} \} \\
  I_{i,j} &= \max \{ M_{i,j-1} - (\rho + \sigma), I_{i,j-1} - \sigma, D_{i,j-1} - (\rho + \sigma) \} \\
  D_{i,j} &= \max \{ M_{i-1,j} - (\rho + \sigma), D_{i-1,j} - \sigma, I_{i-1,j} - (\rho + \sigma) \}
\end{aligned}
$$

- **状态解释**：从 $M$ 转移到 $I$ 或 $D$ 需要支付“开启+延长”总代价；而从 $I$ 继续留在 $I$（或 $D$ 到 $D$）只需支付低廉的延长代价 $\sigma$。

## 3. 生物学意义

仿射罚分模型是现代生物信息学工具（如 BWA, BLAST, ClustalW）的标准配置。它能更准确地识别：
- **同源蛋白中的保守结构域**。
- **基因组重排或大型 Indel 事件**。
- **cDNA 与基因组比对中的长内含子跳跃**。
