这一部分汇总 BioInfo Wiki 的方法主干：先理解序列如何表示、如何被索引，再进入比对、组装和概率模型。

## 这一部分在全站中的位置

它位于“基础与数学”和“分析方向与案例”之间，是整站最稳定的方法层。

## 为什么重要

如果没有这一层，工作流会只剩工具名和命令；而如果只停留在这一层，又很难理解这些方法最终如何落到 DNA-seq、RNA-seq 和宏基因组等真实任务上。

## 推荐阅读顺序

1. [序列表示与索引](../sequence/index.md)
2. [序列比对](../alignment/index.md)
3. [组装与图算法](../assembly/index.md)
4. [概率模型与模式识别](../models/index.md)
