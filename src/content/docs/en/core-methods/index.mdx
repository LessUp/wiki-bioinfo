Bioinformatics algorithms converge on four core methods. Understanding these pillars allows you to navigate the field's diverse tools and applications.

## The Four Pillars

<CardGrid>
  <Card title="1. Indexing & Search" icon="magnifier">
    From k-mer hashing to suffix arrays and FM-index, enabling fast lookup in massive reference sequences.
    
    **Key structures**: Hash tables, Suffix Trees, BWT, FM-index
  </Card>
  <Card title="2. Sequence Alignment" icon="layers">
    Dynamic programming for edit distance, global/local alignment, and heuristic acceleration.
    
    **Key algorithms**: Needleman-Wunsch, Smith-Waterman, BLAST, BWA
  </Card>
  <Card title="3. Genome Assembly" icon="diagram">
    Graph algorithms for reconstructing genomes from short reads—OLC, de Bruijn graphs, and traversal.
    
    **Key algorithms**: Hamiltonian paths, Eulerian paths, graph cleaning
  </Card>
  <Card title="4. Probabilistic Models" icon="open-book">
    HMMs, EM algorithm, and Bayesian inference for handling uncertainty in biological data.
    
    **Key models**: HMM, Profile HMM, PWM/PSSM, Gibbs Sampling
  </Card>
</CardGrid>

## Method Connections

These four pillars are not isolated—they connect in real bioinformatics pipelines:

1. **Read Mapping**: Indexing + Alignment → BWA, Bowtie2
2. **Variant Calling**: Alignment + Probabilistic Models → GATK, DeepVariant
3. **Transcript Assembly**: Alignment + Graph Algorithms → StringTie, Cufflinks
4. **Gene Prediction**: Probabilistic Models + Sequence Analysis → GlimmerHMM, Augustus
