Welcome to BioInfo Wiki, a systematic knowledge base for bioinformatics designed to bridge the gap between tool tutorials and academic papers.

## What Makes This Different

Bioinformatics resources typically fall into two categories:

- **Tool Tutorials** — Tell you how to run commands, but not the underlying principles
- **Academic Papers** — Assume you already understand the background knowledge

BioInfo Wiki fills this gap:

> **Starting from biological objects (DNA, reads, reference genomes), using mathematical and algorithmic language to reformulate problems, then connecting to real analysis pipelines.**

Not a glossary of terms, but structured around **Objects → Models → Algorithms → Pipelines → Resources**, helping you build a navigable knowledge map.

### Core Topics

- **Sequence Alignment**: Needleman-Wunsch, Smith-Waterman, BLAST, minimap2
- **Genome Assembly**: de Bruijn graph, OLC, repeat handling
- **Variant Detection**: Small variants, structural variants, variant annotation and filtering
- **Transcriptomics**: RNA-seq, differential expression, single-cell analysis
- **Probabilistic Models**: HMM, EM algorithm, Bayesian inference
- **Indexing Structures**: Suffix Tree/Array, FM-index, BWT
