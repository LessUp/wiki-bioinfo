## 2. 关键特性：方向性与互补性

### 5' $\to$ 3' 方向
DNA 和 RNA 链具有极性。在生物信息学中，除非特别说明，**字符串默认按 5' 端到 3' 端的顺序存储**。

### 反向互补 (Reverse Complement)

由于 DNA 是双链且反向平行的，对于序列 $S = \text{5'-ATGC-3'}$：

1.  **互补 (Complement)**：按 A-T, G-C 配对规则得到 `3'-TACG-5'`。
2.  **反向 (Reverse)**：将字符串前后颠倒。
3.  **反向互补**：得到 `5'-GCAT-3'`。

- **算法挑战**：在搜索 Motif 或比对 Reads 时，必须同时考虑原始序列及其反向互补链，因为测序仪可能从任何一条链上读取信息。

## 3. 坐标系统：生物信息学的“巨坑”

不同工具和文件格式对位置的计数规则不同，这常导致 **Off-by-one (差一)** 错误。

| 特性 | 0-based (零基) | 1-based (一基) |
| :--- | :--- | :--- |
| **起始编号** | 从 0 开始计算 | 从 1 开始计算 |
| **区间表示** | 通常为半开区间 $[start, end)$ | 通常为闭区间 $[start, end]$ |
| 示例格式 | BED, BAM, Python 索引 | VCF, GFF, SAM (文本), R |
| **长度计算** | $end - start$ | $end - start + 1$ |

### 示例
对于序列 `ACTG`：
- 在 **0-based** 系统中，第一个碱基 `A` 的位置是 0，区间 `[0, 2)` 代表 `AC`。
- 在 **1-based** 系统中，第一个碱基 `A` 的位置是 1，区间 `[1, 2]` 代表 `AC`。

## 4. 字符串抽象的威力

一旦序列被视为字符串，我们就能借用计算机科学中数十年的研究成果：
- **子串匹配**：寻找 Motif 或引物位置。
- **编辑距离**：衡量两条序列之间的变异（替换、插入、缺失）。
- **哈希与索引**：在 30 亿碱基的基因组中实现秒级搜索。
