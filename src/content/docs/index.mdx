## 为什么看这个项目

<CardGrid>
  <Card title="知识主干优先" icon="open-book">
    从对象、模型和算法出发组织内容，而不是只罗列工具命令。
  </Card>
  <Card title="算法与流程相连" icon="puzzle">
    强调字符串、图、概率模型与真实分析流程之间的联系。
  </Card>
  <Card title="中文连续阅读" icon="translate">
    以简体中文为主，术语首次出现补充英文与缩写。
  </Card>
  <Card title="适合反复回查" icon="seti:markdown">
    通过交叉链接把概念、格式、资源和流程串联起来。
  </Card>
</CardGrid>
