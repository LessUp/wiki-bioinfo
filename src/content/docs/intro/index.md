BioInfo Wiki 是一个面向中文社区的开源生物信息学知识库。

它参考 OI Wiki 的组织方式，同时配合一套统一的配图规范：

<figure>
  <img src="/wiki-bioinfo/img/figures/wiki-knowledge-map.png" alt="BioInfo Wiki global knowledge map showing Foundations, Core Methods, Applications, Data & References, and Workflows" />
  <figcaption>BioInfo Wiki 的知识地图：从基础对象和方法，到数据与资源，再到分析方向与完整工作流。</figcaption>
</figure>

- 用清晰的层级结构串起知识点；
- 用统一的文档模板和高信息密度配图降低阅读和贡献门槛；
- 用 GitHub 协作机制持续积累内容。

但它的主题聚焦于 **生物信息学**，目标是把以下几类知识连接起来：

1. 生物学对象：DNA、RNA、蛋白质、基因组、转录组；
2. 算法模型：字符串、图、动态规划、概率模型、索引结构；
3. 数据与资源：参考基因组、注释、文件格式、数据库对象；
4. 分析方向：DNA-seq、RNA-seq、宏基因组、系统发育等真实任务。

## 这个板块在全站中的位置

“开始这里”承担的是整站的 onboarding 入口：帮助新读者先建立知识地图，再决定要沿着教材主线、分析任务还是数据资源主线往下走。

## 为什么要把它单独做成入口

如果项目介绍、学习路线、贡献说明和写作规范都直接混在知识主干里，读者会同时看到“怎么读”和“读什么”，心智负担会比较重。

因此这一板块的职责是：

- 介绍项目想解决什么问题；
- 给出学习路线和推荐入口；
- 提供贡献与写作约定；
- 帮助读者理解全站结构，而不是马上进入过细目录。
