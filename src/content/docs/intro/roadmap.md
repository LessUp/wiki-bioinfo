## 如何选路线

这个项目不是从单一入口强推所有读者，而是提供三条常用路径：先建立对象与流程概念、先抓住算法主线，或者从具体分析任务倒推方法。

## 路线一：初学者路径

适合刚接触生物信息学、希望先建立整体框架的读者。

推荐顺序：

1. [导论](./)
2. [基础与数学](../foundations/)
3. [数据与资源](../data-references/)
4. [分析方向](../applications/)

## 路线二：算法路径

适合有计算机或数学基础、希望先抓住方法主干的读者。

推荐顺序：

1. [算法入门](./algorithms-intro/)
2. [基础与数学](../foundations/)
3. [核心方法](../core-methods/)
4. [分析方向](../applications/)

## 路线三：任务驱动路径

适合已经有具体分析问题，希望边做边回溯原理的读者。

推荐顺序：

1. [分析方向](../applications/)
2. [工作流](../workflows/)
3. [数据与资源](../data-references/)
4. [核心方法](../core-methods/)
5. [基础与数学](../foundations/)

## 最小阅读集合

如果你只想先建立一个可工作的知识框架，建议优先读：

- [生物信息学中的基础对象](../foundations/biology-basics/)
- [参考基因组与注释](../foundations/reference-and-annotation/)
- [常见文件格式](../formats/common-file-formats/)
- [编辑距离](../alignment/edit-distance/)
- [FM-index](../sequence/fm-index/)
- [RNA-seq 工作流](../workflows/rna-seq/)
