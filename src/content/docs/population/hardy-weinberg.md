## 问题引入

1908年，英国数学家 G. H. Hardy 在阅读一篇关于孟德尔遗传的论文时，注意到一个被忽视的关键问题。当时的生物学家争论：如果一个群体中显性等位基因的频率是 70%，隐性等位基因是 30%，经过一代随机交配后，三种基因型的比例会是多少？

一些人认为显性性状会很快占据主导，另一些人则认为复杂的交配模式会让计算变得困难。Hardy 用简短的数学证明了一个反直觉的结论：**在理想条件下，等位基因频率和基因型频率将在代际间保持不变。**

几乎同时，德国医生 Wilhelm Weinberg 独立发表了相同的发现。这个简洁而深刻的定理后来被称为 Hardy–Weinberg 平衡（HWE），成为群体遗传学的第一块基石。


## 数学推导

### 基本设定

考虑一个双等位基因位点，等位基因为 $A$ 和 $a$。设当前群体中：
- $f(AA) = P$，$f(Aa) = Q$，$f(aa) = R$
- 等位基因频率：$p = f(A) = P + \frac{Q}{2}$，$q = f(a) = R + \frac{Q}{2}$

显然 $p + q = 1$。

### 随机交配下的配子频率

在随机交配假设下，每个个体产生的配子中：
- $AA$ 个体只产生含 $A$ 的配子
- $aa$ 个体只产生含 $a$ 的配子  
- $Aa$ 个体产生含 $A$ 或 $a$ 的配子各一半

因此，配子库中：
$$f(A) = P \cdot 1 + Q \cdot \frac{1}{2} + R \cdot 0 = P + \frac{Q}{2} = p$$

同理，$f(a) = q$。

### 下一代基因型频率

随机交配相当于从配子库中独立抽取两个配子。因此下一代基因型频率为：

$$\begin{aligned}
f'(AA) &= p \times p = p^2 \\
f'(Aa) &= 2 \times p \times q = 2pq \\
f'(aa) &= q \times q = q^2
\end{aligned}$$

关键观察：$f'(A) = p^2 + \frac{2pq}{2} = p(p + q) = p$，即等位基因频率保持不变！


## Hardy–Weinberg 平衡的深层含义

### 平衡的本质

HWE 揭示了一个重要事实：**只要满足随机交配的条件，无论初始基因型频率如何，一代之后就会达到平衡状态 $p^2 : 2pq : q^2$，并且此后保持不变。**

这意味着：
- 显性性状的频率不会无限增长
- 隐性等位基因不会因被"掩盖"而消失
- 孟德尔遗传本身不会导致群体组成的变化

### 理想条件假设

严格意义上的 HWE 需要以下假设：
1. **无限大的群体**：排除遗传漂变的影响
2. **随机交配**：排除选型交配和近亲繁殖
3. **无突变**：等位基因不会相互转化
4. **无选择**：所有基因型有相同的生存和繁殖能力
5. **无迁移**：群体是封闭的

真实群体当然不满足所有这些条件——但这恰恰是 HWE 的实用价值所在：**偏离平衡往往暗示着某些进化力量或技术因素在起作用**。

## 统计检验方法

### 卡方检验

最常用的是 Pearson 卡方检验。设观察到的基因型计数为 $n_{AA}$、$n_{Aa}$、$n_{aa}$，总样本量 $N = n_{AA} + n_{Aa} + n_{aa}$。

首先估计等位基因频率：

$$\hat p = \frac{2n_{AA} + n_{Aa}}{2N}, \quad \hat q = 1 - \hat p$$

然后计算期望计数：

$$\begin{aligned}
E_{AA} &= N \hat p^2 \\
E_{Aa} &= 2N \hat p \hat q \\
E_{aa} &= N \hat q^2
\end{aligned}$$

卡方统计量：

$$\chi^2 = \sum_{i} \frac{(O_i - E_i)^2}{E_i}$$

自由度为 1（因为估计了 $p$ 一个参数）。

### 精确检验

对于稀有变异或样本量较小的情况，卡方近似可能不准确。此时可使用精确检验，直接计算在 $H_0$ 下观察到当前或更极端基因型分布的概率。

精确检验特别适用于：
- 低 MAF（次要等位基因频率）位点
- 小样本研究
- 病例对照研究中病例组的选择相关位点


## GWAS 中的质量控制应用

### 对照组的 HWE 过滤

在 GWAS 中，HWE 检验是变异级别质控的关键步骤。标准流程是对**对照组样本**（而非病例组）进行 HWE 检验：

- 对照组偏离 HWE 往往暗示技术问题（分型错误、比对问题、重复区域）或群体分层；
- 常用阈值 $p < 10^{-6}$ 或更严格，过滤显著偏离的位点。

### 病例组的特殊情况

病例组中的疾病相关位点本身就可能偏离 HWE（例如显性遗传病导致 $AA$ 和 $Aa$ 频率升高）。因此：
- **不应对病例组施加严格的 HWE 过滤**；
- 病例组的 HWE 偏离可能是真实信号，而非噪声。

### 偏离原因的诊断

| 偏离模式 | 可能原因 |
|---------|---------|
| 纯合子 excess（$AA$ 和 $aa$ 过多）| 近亲繁殖、群体分层、技术重复 |
| 杂合子 excess | 杂合子优势（平衡选择）、样本污染 |
| 特定纯合子缺失 | 隐性致死、负选择 |
| 系统性偏离 | 分型平台偏差、参考基因组问题 |

## 常见误区与注意事项

### 误区一：偏离 HWE 就是数据错误

偏离 HWE 并不总是技术问题的信号。真实生物学因素同样会导致偏离：
- 近期群体 bottleneck 或扩张事件
- 亚群结构导致的 Wahlund 效应
- 自然选择（正选择、负选择、平衡选择）
- 选型交配

正确解读需要结合上下文，而非机械过滤。

### 误区二：病例组也应满足 HWE

这是 GWAS 质控中最常见的错误之一。与疾病相关的位点在病例组中偏离 HWE 是完全预期的：
- 若疾病由显性等位基因引起，病例组中杂合子和显性纯合子会富集；
- 若疾病由隐性等位基因引起，病例组中隐性纯合子会富集。

因此，HWE 过滤通常仅应用于对照组或全部样本（用于发现系统性问题）。

### 误区三：卡方检验适用于所有位点

对于 $E_i < 5$ 的位点，卡方检验的近似效果不佳。在 GWAS 中，这对应于：
- MAF < 1% 的罕见变异
- 小样本研究

此时应优先使用精确检验。


## 与其他概念的联系

HWE 是理解更复杂群体遗传学现象的基础：

- **连锁不平衡**：HWE 在位点内的应用，LD 是位点间的扩展；
- **群体结构**：亚群混合会导致整体的 Wahlund 效应（杂合子不足）；
- **遗传漂变**：HWE 假设无限大群体，漂变描述小群体的随机波动；
- **近亲繁殖系数 $F$**：$F = 1 - \frac{H_{obs}}{H_{exp}}$，量化偏离 HWE 的程度。
