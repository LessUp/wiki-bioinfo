## 是什么

群体遗传学是研究等位基因频率在群体中如何分布、变化以及被进化力量（自然选择、遗传漂变、迁移、重组）塑造的学科。

它关注的问题包括：

- **等位基因频率**：某个变异在群体中的分布；
- **Hardy–Weinberg 平衡**：在理想条件下基因型频率的预期分布；
- **连锁不平衡（LD）**：不同位点之间等位基因的非随机关联；
- **群体结构**：群体中的亚群分化、祖源和混合历史；
- **自然选择**：有利变异如何扩散，有害变异如何被清除；
- **遗传漂变**：随机采样导致的频率波动。

在生物信息学应用中，群体遗传学是：

- **GWAS 的基础**：理解关联检验、多重检验校正和群体分层；
- **变异解释的背景**：判断变异频率是否异常、是否与疾病相关；
- **进化分析的框架**：理解物种分化、适应性进化等。

## 为什么重要

### 变异解释需要群体背景

单个位点的变异频率不能孤立解释：

- 一个罕见变异可能在某些人群中相对常见（人群特异性）；
- 变异致病性需要考虑其在人群中的频率和分布；
- 携带者频率（carrier frequency）与疾病发病率的关系需要群体遗传学模型。

### GWAS 假阳性控制

GWAS 最大的挑战之一是群体结构导致的假阳性：

- 如果病例和对照来自不同祖源群体，某些位点的频率差异可能反映祖源而非疾病；
- 需要 PCA、线性混合模型等方法校正群体分层；
- Hardy–Weinberg 平衡偏离也是质量控制的重要指标。

### Fine-mapping 的基础

连锁不平衡决定了变异之间的相关性：

- LD block 内的多个变异高度相关，难以区分因果变异；
- Tag SNP 策略利用 LD 减少需要检测的位点数；
- Fine-mapping 需要理解 LD 结构和重组热区。

### 进化机制理解

群体遗传学帮助理解：

- 为什么某些变异在不同人群中频率差异巨大；
- 选择压力如何塑造基因组（如清除选择、平衡选择）；
- 人类迁徙历史和混合事件如何影响现代人群的遗传结构。

## 核心概念

### Hardy–Weinberg 平衡

在理想条件下（无突变、无选择、无限大群体、随机交配），基因型频率会达到平衡：

$$P(AA) = p^2, P(Aa) = 2pq, P(aa) = q^2$$

其中 $p$ 和 $q$ 是等位基因频率。

偏离平衡可能提示：

- **非随机交配**：如近亲繁殖、亚群结构；
- **选择压力**：有利或有害变异的频率偏离预期；
- **突变或漂变**：小群体中的随机波动。

### 连锁不平衡（LD）

LD 衡量两个位点之间的等位基因是否非随机关联：

- **D'**：标准化的 LD 系数，范围 0-1；
- **r²**：相关系数，常用于 GWAS 的 tag SNP 选择；
- **Haplotype block**：LD 较高的区域，重组事件较少。

LD 的影响：

- **Tag SNP**：可以用少数 SNP 代表整个 LD block；
- **Fine-mapping 困难**：高 LD 区域难以确定因果变异；
- **重组热区**：LD 较低的区域，重组频繁。

### 群体结构

群体中的亚群分化导致：

- **等位基因频率差异**：不同亚群间频率不同；
- **PCA 分离**：主成分分析可以识别亚群；
- **混合事件**：不同祖源群体的混合产生新的群体。

检测方法：

- **PCA**：主成分分析识别遗传结构；
- **ADMIXTURE**：估计个体的祖源成分；
- **Fst**：衡量亚群间的分化程度。

## 标准分析流程

### 1. 数据准备

- **基因型数据**：PLINK 格式（.bed/.bim/.fam）或 VCF；
- **质量控制**：去除低质量样本和位点（缺失率、MAF、HWE）；
- **参考数据**：如 1000 Genomes、gnomAD 作为频率参考。

### 2. 基础统计

- **等位基因频率**：计算每个位点的频率；
- **Hardy–Weinberg 检验**：识别偏离平衡的位点；
- **LD 计算**：计算位点间的 D' 和 r²。

### 3. 群体结构分析

- **PCA**：识别主要遗传成分；
- **ADMIXTURE**：估计祖源成分；
- **系统发育树**：构建样本间的遗传关系。

### 4. 关联分析（GWAS）

- **关联检验**：线性/逻辑回归检验基因型-表型关联；
- **多重检验校正**：控制 FDR 或使用 Bonferroni；
- **群体校正**：加入 PCA 主成分或使用混合模型。

## 输入输出

### 输入

- **基因型数据**：PLINK 格式或 VCF；
- **表型数据**：定量性状、二分类性状等；
- **协变量**：年龄、性别、PC 主成分等；
- **参考数据**：用于频率比较或祖源估计。

### 输出

- **等位基因频率**：每个位点的频率；
- **LD 矩阵**：位点间的 LD 关系；
- **PCA 坐标**：样本的主成分坐标；
- **关联结果**：p-value、效应大小、置信区间；
- **祖源估计**：个体的祖源成分比例。

## 常见误区

### Hardy–Weinberg 平衡被违反就是错的

不对。实际群体中，HWE 偏离很常见，可能反映选择、漂变或亚群结构。需要结合上下文解释。

### LD 越高越好

不对。高 LD 虽然有利于 tag SNP 策略，但会使 fine-mapping 困难，难以确定因果变异。

### PCA 前几个主成分一定反映生物学意义

不对。PCA 主成分可能反映技术批次、样本污染等问题，需要结合元数据验证。

### GWAS 显著位点就是致病变异

不对。GWAS 发现的是关联位点，受 LD 影响，真正的因果变异可能在附近。

### 群体结构校正后就完全没有假阳性

不对。即使校正了群体结构，其他混杂因素（如批次、环境）仍可能导致假阳性。
