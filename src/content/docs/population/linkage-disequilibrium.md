## 从独立到关联：问题的提出

Hardy–Weinberg 平衡告诉我们，在随机交配的理想群体中，**单个位点**上的等位基因频率保持稳定。但现实世界更复杂：如果两个位点在同一条染色体上，它们是否独立遗传？

考虑两个相邻的 SNP：位点 A（等位基因 A/a）和位点 B（等位基因 B/b）。如果它们相距极近，重组事件很难在两者之间发生。这意味着，如果某位点携带等位基因 A，其附近的位点 B 也很可能携带某个特定的等位基因——这种**非随机关联**就是连锁不平衡（Linkage Disequilibrium, LD）。

LD 是理解基因组结构和 GWAS 信号的关键：它解释了为什么全基因组芯片只需检测约 50-100 万个 tag SNP，就能覆盖绝大部分常见变异的信息。


## 数学定义与度量

### 基本定义

设位点 A 的等位基因频率为 $p_A$ 和 $q_A$，位点 B 为 $p_B$ 和 $q_B$。如果两位点独立，则单倍型 AB 的期望频率应为 $p_A \cdot p_B$。

定义**连锁不平衡系数 $D$**：

$$D = P_{AB} - p_A p_B$$

其中 $P_{AB}$ 是观察到的 AB 单倍型频率。当 $D = 0$ 时，两位点处于连锁平衡状态。

$D$ 的取值范围为 $[-\min(p_A p_B, q_A q_B), \min(p_A q_B, q_B p_A)]$，其边界依赖于等位基因频率。这使得直接比较不同位点对的 $D$ 值变得困难。

### 标准化度量：D'

为解决 $D$ 的频率依赖问题，Lewontin (1964) 提出了标准化的 $D'$：

$$D' = \begin{cases}
\frac{D}{\min(p_A p_B, q_A q_B)} & D > 0 \\
\frac{D}{\min(p_A q_B, q_B p_A)} & D < 0
\end{cases}$$

$D'$ 的取值范围为 $[-1, 1]$，但其物理意义更直观的是其绝对值 $|D'| \in [0, 1]$：
- $|D'| = 1$ 表示完全连锁不平衡（四位单倍型中至少有一种缺失）；
- $|D'| = 0$ 表示连锁平衡。

### 相关系数：r²

GWAS 中最常用的度量是皮尔逊相关系数的平方：

$$r^2 = \frac{D^2}{p_A q_A p_B q_B}$$

$r^2$ 的直观解释：
- 它表示用一个位点预测另一个位点的准确度；
- 在病例对照研究中，若因果位点未被直接检测，$r^2$ 决定了 tag SNP 能捕获多少关联信号；
- $r^2 = 1$ 表示两位点完全等价（在统计上不可区分）。


## LD 的生物学起源

### 重组的限制

LD 的根本来源是**重组的历史限制**。当两个位点在染色体上距离较近时，世代间的减数分裂重组难以将它们打散。重组率 $c$ 与物理距离相关，LD 的预期衰减可以用以下关系描述：

$$E[r^2] \approx \frac{1}{1 + 4N_e c}$$

其中 $N_e$ 是有效群体大小。这表明：
- 大群体中 LD 衰减更快（更多重组机会）；
- 重组热点区域 LD 迅速下降；
- 重组冷区或倒置区域 LD 得以长期维持。

### 群体历史的影响

除重组外，群体历史事件也塑造 LD 模式：

| 事件 | 对 LD 的影响 |
|-----|------------|
| 瓶颈效应 | 减少有效 $N_e$，增强 LD |
| 群体扩张 | 增加 $N_e$，加速 LD 衰减 |
| 混合事件 | 引入新的单倍型组合，局部降低 LD |
| 选择 | 有利单倍型被保留，导致 LD 增强 |

### 突变与单倍型的诞生

每个突变事件都发生在一个特定的染色体背景（单倍型）上。在突变刚出现时，它与周围位点处于完全的 LD 状态（$r^2 = 1$）。随着时间推移，重组逐渐打破这种关联，LD 衰减速度与位点间距成正比。


## Haplotype Block 结构

### 块状的 LD 景观

人类基因组的 LD 并非均匀分布，而是呈现**块状结构**（haplotype block）。在一个 block 内，SNP 之间高度相关；block 之间则相对独立。

这种结构源于：
- **重组热点**：某些基因组区域重组率极高，成为 block 边界；
- **重组冷区**：大部分区域重组率低，维持长距离 LD；
- **历史单倍型**：人类群体历史中的有限数量的祖先单倍型。

### Block 的识别与表示

常用的 block 定义方法包括：
- **D' 置信区间法**：Gabriel 等 (2002) 基于 $D'$ 的置信区间定义 block；
- **四配子检验**：Wang 等 (2002) 通过检测是否四种可能单倍型都存在来定义 block；
- **热图可视化**：LD heatmap 中，高 LD block 表现为对角线上的深色三角。


## 生物信息学应用

### GWAS 中的 LD 考虑

**关联信号的扩散**：GWAS 发现的显著位点往往并非因果变异，而是与其处于高 LD 的代理 SNP。显著信号通常表现为一个"峰"，包含多个高 $r^2$ 的 SNP。

**精细定位（Fine-mapping）**：利用 LD 结构缩小候选因果变异集合。若某变异与显著位点的 $r^2$ 很低，则不太可能是真正的因果位点。

### Tag SNP 策略

基于 LD 的冗余性，可以用少量 tag SNP 代表整个 block 的变异信息：
- HapMap 和 1000 Genomes 项目通过系统性的 LD 分析筛选 tag SNP；
- 商业芯片（如 Illumina、Affymetrix）基于此策略设计探针；
- 典型的全基因组芯片只需 50-100 万 tag SNP 即可覆盖大部分常见变异。

### 基因型插补（Imputation）

利用参考群体（如 1000 Genomes）的 haplotype 结构和 LD 模式，可以推断未直接检测的位点基因型：
- 基于隐马尔可夫模型（如 IMPUTE2、Beagle）；
- LD 结构是插准确性的关键决定因素；
- 允许用低密度芯片数据推断全基因组变异。

### 群体历史推断

LD 模式携带群体历史信息：
- **LD 衰减曲线**：反映有效群体大小随时间的变化；
- **单倍型长度分布**：可用于推断近期的选择事件；
- **跨群体 LD 差异**：揭示混合历史和分化时间。


## 常见误区

### 误区一：显著 SNP 就是因果 SNP

GWAS 显著位点很可能只是与因果变异处于高 LD 的 tag SNP。在欧洲人群中，因果变异与最显著位点的平均距离可达数十 kb。精细定位需要结合功能注释和跨群体的 LD 差异。

### 误区二：LD 是固定不变的属性

LD 是**群体特异性**的：
- 不同祖源群体有不同的 LD 结构（如非洲人群 LD 衰减更快）；
- 混合群体的 LD 模式反映了其祖源群体的加权组合；
- 比较不同群体的 LD 可以揭示选择信号。

### 误区三：物理距离近就一定高 LD

虽然 LD 通常随距离衰减，但：
- 重组热点可以打破邻近位点的 LD；
- 倒置多态性导致某些区域 LD 异常高；
- 结构性变异（如缺失、重复）会改变局部 LD 模式。


## 与其他概念的联系

- **Hardy–Weinberg 平衡**：HWE 描述位点内的基因型频率，LD 扩展到位点间的关联；
- **重组**：LD 的衰减是重组的累积效应，两者互为镜像；
- **群体结构**：亚群分化会产生长距离 LD（admixture LD），与物理连锁的 LD 机制不同；
- **选择**：正选择会导致选择性清扫（selective sweep），在目标位点周围产生高 LD 区域。
