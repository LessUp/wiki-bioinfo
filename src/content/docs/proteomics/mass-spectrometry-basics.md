## 是什么

基于质谱的蛋白质组学通常不是直接测整条蛋白，而是：

1. 先把蛋白酶解成 peptide；
2. 将 peptide 电离后送入质谱仪；
3. 测量它们的 **m/z（mass-to-charge ratio）**；
4. 再通过碎裂谱推断肽段序列。

## MS1 与 MS2

### MS1（一级质谱）

第一次扫描记录的是完整 peptide 离子的 m/z 和峰强度，给出候选 precursor。

**作用**：
- 识别可能的肽段前体离子
- 测量肽段的完整分子量
- 提供定量信息（峰强度）

### MS2（二级质谱）

选中某个 precursor 后将其碎裂，再测碎片离子的 m/z，得到碎裂谱。这个谱图是 peptide 鉴定的核心证据。

**碎裂方式**：
- **CID**（碰撞诱导解离）：最常用的碎裂方法
- **HCD**（高能碰撞解离）：更高效的碎裂
- **ETD**（电子转移解离）：适用于大肽段和修饰肽段

## 为什么是 peptide 而不是直接 protein

因为整条蛋白过长、结构复杂，不适合直接做高通量高分辨率测量；而 peptide 更容易：

- **分离**：通过液相色谱分离
- **电离**：肽段比完整蛋白更容易带上电荷
- **解释**：碎裂模式更可预测，便于序列推断

## 肽段测序问题

### 问题定义

**Peptide Sequencing Problem**：给定实验测得的质谱（碎片离子质量列表），重建产生该谱图的肽段序列。

**输入**：
- 母体肽段质量 $M$
- 碎片离子质量的多重集合 $\{m_1, m_2, ..., m_n\}$

**输出**：氨基酸序列 $s = s_1s_2...s_k$，使得：
- 序列总质量 $\sum mass(s_i) = M$
- 序列产生的理论碎片质量与实验谱匹配

### 氨基酸质量表

常见氨基酸残基质量（Da）：

| 氨基酸 | 单字母 | 残基质量 |
|--------|--------|----------|
| Glycine | G | 57.02 |
| Alanine | A | 71.04 |
| Valine | V | 99.07 |
| Leucine | L | 113.08 |
| Serine | S | 87.03 |
| Threonine | T | 101.05 |
| ... | ... | ... |

## 谱图（Spectrum Graph）

### 概念

**谱图**是把质谱数据转换成图结构的计算表示，节点代表质量峰，边代表可能的氨基酸残基。

**构建方法**：
1. 每个质量峰对应图中的一个节点
2. 如果两个峰的质量差等于某个氨基酸残基质量，则添加一条带标签的边
3. 添加源节点（质量=0）和汇节点（质量=M）

### 示例

假设母体质量 M = 1000，测得碎片峰：{0, 57, 114, 201, 303, 400, 500, 600, 700, 800, 900, 1000}

可能的边：
- 0 → 57（Glycine，质量差 57）
- 57 → 114（Glycine）
- 114 → 201（Serine，质量差 87）
- ...

### 从谱图到序列

在谱图中，从源到汇的路径对应可能的肽段序列：
- 路径上的边标签拼接成氨基酸序列
- 路径的"质量"等于所有边对应残基质量之和

## 从头测序 vs 数据库搜索

### 从头测序（De Novo Sequencing）

不依赖数据库，直接从谱图推断序列。

**方法**：
- 在谱图中寻找最长路径
- 使用动态规划算法
- 处理噪声和缺失峰

**优势**：
- 可发现新肽段、突变、修饰
- 不依赖数据库完整性

**挑战**：
- 计算复杂度高
- 对谱图质量要求高
- 存在组合爆炸问题

### 数据库搜索

将实验谱与数据库中理论谱比对。

**流程**：
1. 根据蛋白质数据库生成理论肽段
2. 预测每个肽段的理论质谱
3. 与实验谱比对评分
4. 选择最佳匹配

**评分函数**：
- 共享峰数量
- 谱的相关系数
- 概率评分（如 SEQUEST、X!Tandem、Mascot）

## 谱比对（Spectral Alignment）

### 问题背景

实验谱与理论谱很少完全一致，原因：
- 测量误差（质量偏差）
- 肽段修饰（磷酸化、糖基化等）
- 非特异性碎裂

### Spectral Alignment Problem

给定实验谱 $S$ 和理论谱 $S'$，找到使两者"对齐"的偏移和修饰映射。

**核心操作**：
- **匹配**：实验峰与理论峰对应
- **移位**：允许质量偏移（修饰）
- **插入/删除**：处理噪声峰

## 常见误区

- **峰越高就一定是蛋白更多**：还受电离效率、肽段性质和仪器响应影响；
- **测到 peptide 就等于直接测到独特蛋白**：有些 peptide 会在多个蛋白间共享；
- **MS1 就能完成全部鉴定**：很多场景仍需依赖 MS2 碎裂谱；
- **谱图完美匹配才是正确鉴定**：实际谱图总有噪声，需要容忍偏差。
