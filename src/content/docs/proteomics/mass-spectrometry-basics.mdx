## 1. 为什么不直接测序蛋白质？

与 DNA 测序不同，直接对长蛋白质进行测序面临巨大挑战：
- **长度与结构**：蛋白质通常由数百甚至上千个氨基酸组成，且具有复杂的二级和三级结构。
- **化学异质性**：20 种氨基酸的化学性质差异巨大，且存在各种翻译后修饰。
- **动态范围**：细胞内蛋白质浓度的差异可达 10^6 倍以上。

**解决方案**：采用“底向上” (Bottom-up) 策略，先用蛋白酶（如胰蛋白酶 Trypsin）将蛋白质切成短小的**肽段 (Peptides)**，再通过质谱分析这些肽段。

## 2. 串联质谱 (Tandem MS/MS) 流程

质谱仪的工作原理可以简化为两个关键阶段：

### MS1：前体扫描 (Precursor Scan)
测量完整肽段离子的质荷比 ($m/z$)。
- **作用**：确定肽段的分子量，并选择其中一个高丰度的离子进行后续碎裂。

### MS2：碎片扫描 (Fragment Scan)
将选定的肽段离子通过碰撞（如 CID/HCD）碎裂成更小的碎片。
- **作用**：产生反映肽段内部序列信息的碎裂谱。这是肽段测序的核心数据。

## 3. 肽段碎裂的数学模型

当肽段沿骨架碎裂时，会产生两类主要的碎片离子：
- **b-ion**：包含 N 端（氨基端）的碎片。
- **y-ion**：包含 C 端（羧基端）的碎片。

**核心等式**：
对于一个位置的碎裂，其产生的 b-ion 和 y-ion 的质量之和应等于母离子质量加上水分子的质量（约 18 Da）。
- 这种**互补性**是验证谱图真伪和辅助测序的关键直觉。

## 4. 肽段测序问题 (Peptide Sequencing Problem)

**形式化定义**：
给定实验测得的碎片质量多重集 $S$、母离子质量 $M$ 以及离子类型偏移 $\Delta$，寻找一个氨基酸序列 $P$，使其产生的理论谱与 $S$ 的匹配得分最高。

- **寻路直觉**：如果将质量看作顶点，氨基酸质量看作边，那么肽段测序就转化为在**谱图 (Spectrum Graph)** 这个有向无环图中寻找一条从 0 到 $M$ 的最优路径。
