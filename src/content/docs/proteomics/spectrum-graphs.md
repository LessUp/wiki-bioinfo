## 问题背景

### 肽段测序问题

在串联质谱（tandem MS/MS）中，我们获得一个实验质谱图 S，需要推断出产生这个质谱图的肽段序列 P。

**输入**：
- 实验质谱图 S = {s₁, s₂, ..., s_q}
- 离子类型集合 Δ = {δ₁, ..., δ_k}（如 b-ions、y-ions 等）
- 母离子质量 m

**输出**：
- 质量为 m 的肽段 P，使其理论质谱图与实验质谱图 S 的匹配度最高

### 传统方法的局限

**穷举搜索方法**：
- 生成所有 20^l 个长度为 l 的氨基酸序列
- 计算每个序列的理论质谱图
- 找到与实验质谱图匹配最好的序列

**问题**：
- 搜索空间呈指数增长（20^l）
- 即使使用分支定界和前缀剪枝，计算量仍然巨大
- 前缀剪枝可能错误地丢弃正确序列

## 谱图图构建

### 基本思想

谱图图方法不生成所有可能的氨基酸序列，而是从实验质谱图直接构建图结构，将肽段测序问题转化为图中的路径搜索问题。

### 离子类型

质谱仪在碎裂肽段时会产生多种离子类型：

### 图构建步骤

假设我们只考虑 N-terminal 离子，忽略 C-terminal 离子以简化说明。

**步骤1：为每个质谱峰创建顶点**

对于实验质谱图中的每个质量 s ∈ S，由于不知道它是由哪种离子类型产生的，我们为每种离子类型 δ_j 创建一个"猜测"顶点。

如果质量 s 是由离子类型 δ_j 产生的，那么部分肽段的质量 x 满足：
```
s = x - δ_j
x = s + δ_j
```

因此，对于每个 s，我们创建 k 个顶点，标记为质量 s + δ₁, s + δ₂, ..., s + δ_k。

**步骤2：添加起始和终止顶点**

- 起始顶点：质量 0
- 终止顶点：质量 m（母离子质量）

**步骤3：添加边**

在顶点 u 和 v 之间添加有向边 (u, v)，如果：
- mass(v) > mass(u)
- mass(v) - mass(u) 等于某个氨基酸的质量

边用对应的氨基酸标记。

### 图的形式化定义

给定实验质谱图 S = {s₁, ..., s_q} 和离子类型集合 Δ = {δ₁, ..., δ_k}：

- **顶点集合**：V = {s_initial} ∪ V₁ ∪ ... ∪ V_q ∪ {s_final}
  - s_initial = 0
  - s_final = m
  - V_i(s_i) = {s_i + δ₁, ..., s_i + δ_k}（每个质谱峰生成 k 个顶点）
  
- **边集合**：E = {(u, v) | mass(v) - mass(u) = m(a) for some amino acid a}

- **边标签**：label(u, v) = 氨基酸 a，其中 m(a) = mass(v) - mass(u)

谱图图最多有 qk + 2 个顶点。

### Worked Example

假设：
- 质谱图 S = {58, 149, 204, 351, 442}
- 离子类型 Δ = {1}（只考虑 b-ions）
- 母离子质量 m = 498
- 氨基酸质量：G=57, P=97, F=147, N=114, A=71

**构建顶点**：
- 起始顶点：0
- 对于每个 s ∈ S，创建顶点 s + 1：
  - 58 → 59
  - 149 → 150
  - 204 → 205
  - 351 → 352
  - 442 → 443
- 终止顶点：498

**添加边**：
- 0 → 59（边标签 G，因为 59 - 0 = 59 - 1 = 58 ≈ 57）
- 59 → 150（边标签 P，因为 150 - 59 = 91 ≈ 97）
- ... 等等

实际中，由于质量测量有误差，需要允许一定的质量容差。

## 从谱图图到肽段序列

### 完整谱图

**定义**：质谱图 S 是完整的，如果对于肽段 P = p₁...pₙ 的每个 N-terminal 部分肽段 P_i（1 ≤ i ≤ n），S 中至少包含一个对应的离子类型。

**关键观察**：对于完整谱图，谱图中存在一条从 s_initial 到 s_final 的路径，其边标签正好是肽段序列 P。

### 肽段测序作为路径问题

对于完整谱图，肽段测序问题转化为：
- 在有向无环图（DAG）中寻找从 0 到 m 的路径
- 理想情况下，正确路径是边数最多的路径（Longest Path in DAG）

### 不完整谱图的挑战

实验质谱图通常是不完整的：
- 某些离子类型可能缺失
- 存在化学噪声（无关的峰）
- 即使谱图完整，也可能存在多条相同长度的路径

**问题**：单纯选择边数最多的路径不能反映顶点的"重要性"。

## 概率评分方法

### 基本思想

不同离子类型出现的概率不同：
- b-ions 最常见（概率高）
- b-H₂O-NH₃ 等罕见离子（概率低）

因此，应该根据离子类型的概率对顶点进行评分。

### 概率模型

假设：
- 每个离子类型 δ_i 有出现的概率 p(δ_i)
- 不同离子类型的出现相互独立
- 化学噪声产生任意质量的概率为 p_R

对于部分肽段 P_i，如果它产生离子类型 δ₁, ..., δ_l（"存在"的离子），而不产生 δ_{l+1}, ..., δ_k（"缺失"的离子），则顶点得分为：

```
score = [∏_{i=1}^{l} p(δ_i)] × [∏_{i=l+1}^{k} (1 - p(δ_i))] ×
        [p_R / (1 - p_R)]
```

这个评分考虑了：
- 奖励解释顶点的离子类型
- 惩罚缺失的离子类型
- 考虑化学噪声的影响

### 改进的路径选择

使用概率评分后，肽段测序问题转化为：
- 在谱图图中寻找从 0 到 m 的路径
- 最大化路径上顶点评分的总和

这可以通过动态规划在 DAG 中高效求解。

## 算法复杂度

### 图构建

- **时间复杂度**：O(qk)，其中 q 是质谱峰数，k 是离子类型数
- **空间复杂度**：O(qk)

### 路径搜索

- **时间复杂度**：O(V + E) = O(qk + qk·20) ≈ O(qk)，在 DAG 中使用拓扑排序
- **空间复杂度**：O(V) = O(qk)

## 实际应用

### 考虑 C-terminal 离子

上述简化模型只考虑了 N-terminal 离子。实际应用中：
- 将质谱图 S 与其"反转"版本结合
- 同时考虑 N-terminal 和 C-terminal 离子
- 提高肽段测序的准确性

### 质量容差

实际质谱仪测量有误差：
- 允许一定的质量容差（如 ±0.5 Da）
- 在构建边时使用模糊匹配而非精确匹配

### 评分函数的改进

实际工具使用更复杂的评分函数：
- 考虑峰强度（intensity）
- 使用更精确的离子类型概率模型
- 考虑氨基酸的先验概率

## 常见误区

### 谱图图总是能唯一确定肽段序列

不是。即使使用概率评分，谱图图方法也可能：
- 返回多个候选序列
- 对于低质量质谱图，结果不确定
- 需要结合数据库搜索等方法

### 完整谱图在实际中很常见

不是。实验质谱图通常是不完整的：
- 某些离子类型可能完全缺失
- 噪声峰可能干扰路径选择
- 需要概率评分来处理不完整性

### 只需要考虑 b-ions

不是。实际应用中需要：
- 同时考虑多种离子类型
- 利用 N-terminal 和 C-terminal 离子的互补性
- 使用更复杂的离子类型模型

## 与真实工具的连接

谱图图方法在真实工具中的应用：

- **PEAKS**：使用谱图图进行从头肽段测序
- **Novor**：基于深度学习的谱图图方法变种
- **PepNovo**：使用概率谱图图模型
- **Sherenga**：早期的谱图图实现

现代蛋白质组学流程通常：
- 对高质量谱图使用谱图图进行从头测序
- 对低质量谱图使用数据库搜索
- 结合两种方法提高鉴定率
