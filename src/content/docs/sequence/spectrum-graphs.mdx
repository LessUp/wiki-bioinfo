## 是什么

谱图（Spectrum Graph）是解决**肽段测序问题**的图论方法。给定质谱实验数据，目标是重建产生该谱图的肽段氨基酸序列。

## 要解决什么生物信息学问题

### 肽段测序问题

给定实验质谱 $S = \{s_1, s_2, ..., s_q\}$、离子类型集合 $\Delta$ 和母离子质量 $m$，找到质量为 $m$ 的肽段 $P$，使其理论谱与实验谱匹配最优。

**输入**：
- 实验质谱 $S$
- 离子类型集合 $\Delta = \{\delta_1, ..., \delta_k\}$
- 母离子质量 $m$

**输出**：质量为 $m$ 的肽段 $P$，其理论谱与 $S$ 匹配最佳。

### 应用场景

1. **De novo 测序**：不依赖数据库，直接从质谱重建肽段序列
2. **数据库搜索**：验证数据库中肽段与实验谱的匹配
3. **修饰鉴定**：检测磷酸化、糖基化等翻译后修饰

## 核心思想

### 图的构建

对于实验质谱中的每个质量 $s \in S$，生成 $k$ 个可能的顶点（对应 $k$ 种离子类型）：

$$V_i(s) = \{s + \delta_1, s + \delta_2, ..., s + \delta_k\}$$

顶点集合为：
$$V = \{s_{initial}\} \cup V_1 \cup ... \cup V_q \cup \{s_{final}\}$$

其中 $s_{initial} = 0$，$s_{final} = m$（母离子质量）。

### 边的定义

当且仅当两个顶点 $u$ 和 $v$ 的质量差等于某个氨基酸质量时，建立有向边 $(u, v)$：

$$\text{mass}(v) - \text{mass}(u) = \text{mass}(\text{氨基酸})$$

### 路径解释

谱图中从 $s_{initial}$ 到 $s_{final}$ 的路径对应一个候选肽段，边的标签构成氨基酸序列。

## 算法步骤

### 1. 顶点生成

```
对于实验谱中每个质量 s:
    对于每种离子类型 δ:
        创建顶点 v = s + δ
        记录该顶点的来源峰和离子类型
```

### 2. 边构建

```
对于每对顶点 u, v:
    如果 mass(v) - mass(u) ∈ 氨基酸质量集合:
        创建有向边 u → v
        边标签 = 对应氨基酸
```

### 3. 最长路径

对于**完整谱**（包含所有 N-端部分肽段），最长路径即为正确答案：

```
EXONCHAINING 变体:
    初始化: s[0] = 0, s[i] = -∞
    
    对于每个顶点 v_i（按质量排序）:
        对于每条进入边 (v_j → v_i):
            s[i] = max(s[i], s[j] + 1)
    
    返回 s[final] 及回溯路径
```

时间复杂度：$O(|V| + |E|)$，其中 $|V| \leq qk + 2$，$|E|$ 取决于质量匹配。

## 实例说明

### 示例：肽段 GPFNA

假设实验谱测得以下 b-ion 峰（简化）：

| 离子 | 质量 |
|------|------|
| b1 (G) | 58 |
| b2 (GP) | 149 |
| b3 (GPF) | 296 |
| b4 (GPFN) | 410 |
| 母离子 | 498 |

**谱图构建**：

1. **顶点**：0, 58, 149, 296, 410, 498
2. **边**：
   - 0 → 58 (G, 质量差 58)
   - 58 → 149 (P, 质量差 91)
   - 149 → 296 (F, 质量差 147)
   - 296 → 410 (N, 质量差 114)
   - 410 → 498 (A, 质量差 88)

3. **最长路径**：0 → 58 → 149 → 296 → 410 → 498，对应序列 G-P-F-N-A

## 处理不完整谱

实际实验谱通常**不完整**（某些离子缺失），需要改进评分：

### 概率评分模型

为每个顶点赋予分数，反映其可靠性：

$$\text{score}(v) = \sum_{i \in \text{present}} \log\frac{p(\delta_i)}{p_R} + \sum_{i \in \text{missing}} \log\frac{1-p(\delta_i)}{1-p_R}$$

其中：
- $p(\delta_i)$：离子类型 $\delta_i$ 的出现概率
- $p_R$：化学噪声概率

### 带权最长路径

将顶点权重纳入动态规划：

$$s[i] = \max_{j: (j,i) \in E} \{s[j] + w(v_i)\}$$

## 考虑 C-端离子

实际应用中需同时考虑 b-ion（N-端）和 y-ion（C-端）：

**方法**：将实验谱与其"反转"版本结合，分别构建 N-端和 C-端顶点，确保路径兼容性。

## 算法复杂度

| 步骤 | 复杂度 |
|------|--------|
| 顶点生成 | $O(qk)$ |
| 边构建 | $O((qk)^2 \cdot |A|)$，可优化为 $O(qk \cdot |A|)$ |
| 最长路径 | $O(qk + |E|)$ |
| **总体** | **$O(qk \cdot |A|)$** |

$q$ = 实验谱峰数，$k$ = 离子类型数，$|A|$ = 氨基酸种类（20）

## 局限性与改进

### 主要挑战

1. **谱不完整**：某些离子类型可能完全缺失
2. **质量精度**：质谱测量误差导致顶点匹配困难
3. **修饰存在**：翻译后修饰改变氨基酸质量
4. **混合谱**：多个肽段共存产生复杂谱图

### 改进策略

| 问题 | 解决方案 |
|------|----------|
| 质量误差 | 设置质量容差窗口（如 ±0.5 Da） |
| 修饰鉴定 | 扩展边标签包含常见修饰质量 |
| 噪声过滤 | 预处理去除低强度峰 |
| 多条路径 | 保留 top-k 候选路径供后续验证 |

## 历史注记

谱图方法由 **Pavel Pevzner** 等人在 2000 年左右系统发展，成为 de novo 肽段测序的算法基础。早期算法如 PEAKS、PepNovo 均基于谱图框架，后续发展引入了概率模型和机器学习改进顶点评分。

## 总结

- **谱图将质谱数据转化为图结构**，使肽段测序成为图上的路径寻找问题
- **对于完整谱**，最长路径给出正确答案
- **对于不完整谱**，概率评分改进顶点权重，提高鲁棒性
- **是连接实验质谱与序列推断的关键算法桥梁**
