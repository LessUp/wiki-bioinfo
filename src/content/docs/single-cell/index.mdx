## 是什么

单细胞组学是指对单个细胞进行基因组、转录组、表观组或多组学测量的技术总称。

传统 bulk 测序（如 bulk RNA-seq）给出的是样本中所有细胞的平均信号，而单细胞技术能为每个细胞独立建库、测序，揭示：

- **细胞类型鉴定**：识别组织中不同的细胞亚群；
- **细胞状态变化**：捕捉激活、分化、应激等状态转换；
- **发育轨迹推断**：在"伪时间"上重建细胞从前体到终态的分化路径；
- **稀有亚群发现**：检测低丰度的稀有细胞（如干细胞、循环肿瘤细胞）；
- **细胞通讯分析**：通过配体-受体表达推断细胞间信号交流。

常见的单细胞技术包括：

- **scRNA-seq**：单细胞转录组测序，应用最广泛；
- **scATAC-seq**：单细胞染色质可及性测序；
- **scDNA-seq**：单细胞基因组测序；
- **multiome**：同时测量同一细胞的多种组学（如 scRNA-seq + scATAC-seq）。

## 为什么重要

### 细胞异质性

生物系统不是均一的：

- **肿瘤**：同一肿瘤内包含不同克隆、不同状态的癌细胞，以及浸润的免疫细胞、成纤维细胞等；
- **大脑**：神经元和胶质细胞有极其多样的亚型，每个亚型有不同的功能；
- **免疫系统**：T 细胞、B 细胞、巨噬细胞等有复杂的激活状态和分化路径；
- **发育**：胚胎发育过程中，细胞从全能到多能到终态，状态连续变化。

Bulk 平均会掩盖这些异质性，导致错误的生物学结论。

### 发育轨迹与分化

scRNA-seq 可以捕捉发育过程中的连续状态：

- 通过"伪时间"排序，重建分化路径；
- 识别分支点和终末状态；
- 发现新的中间状态或过渡细胞。

这对理解发育机制、干细胞研究和再生医学至关重要。

### 细胞通讯

不同细胞之间通过信号分子交流：

- 配体-受体分析可以推断细胞间的信号网络；
- 理解免疫细胞如何识别和攻击肿瘤；
- 揭示器官发育中的细胞间相互作用。

## 核心技术挑战

### 数据稀疏性

单细胞数据的显著特征是**稀疏**：

- 每个细胞只检测到部分基因（通常 10-30%）；
- 大量基因表达量为 0（dropout）；
- 技术噪声比 bulk 数据大得多。

这导致：

- 需要特殊的归一化和降维方法；
- 差异表达分析需要考虑零膨胀；
- 可视化时需要谨慎解释低表达基因。

### 细胞捕获效率

单细胞实验的捕获效率通常只有 50-80%：

- 某些细胞类型（如大细胞、脆性细胞）更难捕获；
- 捕获偏差可能导致细胞类型比例失真；
- 需要通过实验设计和计算校正来缓解。

### 批次效应

不同批次、不同平台的数据存在系统性差异：

- 即使是同一组织，不同批次的细胞聚类可能分开；
- 需要批次校正方法（如 Harmony、Seurat integration、BBKNN）；
- 过度校正可能抹消真实的生物学差异。

### 高维性

单细胞数据通常是高维的（20,000 基因 × 数千细胞）：

- 直接可视化困难，需要降维（PCA、t-SNE、UMAP）；
- 聚类算法需要处理高维稀疏数据；
- 计算资源需求大。

## 标准分析流程

### 1. 实验设计

关键考虑：

- **细胞数量**：根据研究目标确定（通常 1,000-10,000 细胞）；
- **捕获平台**：10x Genomics、Drop-seq、SMART-seq2 等；
- **样本类型**：新鲜组织、冻存组织、培养细胞；
- **批次设计**：是否需要技术重复、生物学重复。

### 2. 数据预处理

包括：

- **Read 比对**：将 reads 比对到参考基因组；
- **Barcode/UMI 去重**：识别细胞和分子；
- **Count matrix 生成**：生成基因 × 细胞的表达矩阵；
- **质量控制**：过滤低质量细胞（低基因数、高线粒体比例）；
- **归一化**：消除测序深度和细胞大小差异。

### 3. 降维与聚类

- **特征选择**：选择高变基因（HVG）；
- **PCA**：降维到主成分空间；
- **聚类**：Leiden、Louvain 等算法识别细胞亚群；
- **可视化**：UMAP、t-SNE 二维可视化。

### 4. 细胞类型注释

通过以下方式识别细胞类型：

- **标记基因**：已知细胞类型的特异性基因；
- **参考数据库**：如 CellMarker、PanglaoDB；
- **自动注释工具**：如 SingleR、scMAP；
- **人工验证**：结合领域知识确认。

### 5. 差异表达与功能分析

- **差异表达**：比较不同细胞群之间的基因表达差异；
- **通路富集**：GO、KEGG 等通路分析；
- **基因集评分**：如 GSVA、AUCell 评估特定通路活性。

### 6. 轨迹推断（可选）

- **伪时间排序**：Monocle、Slingshot、PAGA；
- **RNA velocity**：推断细胞的动态变化方向；
- **分支点识别**：识别分化决策点。

## 输入输出

### 输入

- **FASTQ 文件**：测序原始数据；
- **参考基因组**：用于比对；
- **样本元数据**：实验条件、批次信息等。

### 输出

- **Count matrix**：基因 × 细胞的表达矩阵；
- **细胞聚类结果**：每个细胞的簇标签；
- **降维坐标**：PCA、UMAP 坐标；
- **细胞类型注释**：每个细胞的类型标签；
- **差异表达基因列表**；
- **可视化图表**：聚类图、热图、轨迹图等。

## 常见误区

### Dropout = 表达量为 0

不对。单细胞数据中大量的 0 值很多是技术性 dropout（未检测到），而不是真正的无表达。需要考虑零膨胀和归一化。

### 细胞聚类结果就是生物学真相

不对。聚类结果高度依赖于参数选择（如分辨率、邻居数），需要结合标记基因和生物学知识验证。

### UMAP/t-SNE 距离有生物学意义

不对。UMAP/t-SNE 是非线性降维方法，主要用于可视化，距离和密度不一定反映真实的生物学关系。

### 细胞数量越多越好

不对。细胞数量增加会带来成本和计算负担，需要根据研究目标平衡。对于异质性低的样本，过多细胞可能不提供额外信息。

### 批次校正后就可以直接合并数据

不对。批次校正可能过度校正，抹消真实的生物学差异。需要谨慎选择校正方法和参数，并在校正后验证生物学信号是否保留。
