## 1. 从快照到过程：伪时间的诞生

在发育生物学中，细胞的分化是一个连续过程。但在实验中，我们一次性杀死了所有细胞进行测序。
- **挑战**：我们无法追踪同一个细胞随时间的变化。
- **假设**：如果群体中存在处于不同发育阶段的细胞，那么通过计算它们之间的转录组相似性，我们可以将它们排序。
- **伪时间 (Pseudotime)**：定义为细胞沿发育路径相对于起始点的距离。

## 3. RNA Velocity：向量场的力量

传统的轨迹推断是“回溯式”的，而 **RNA Velocity** 提供了“预测性”的视角。

### 核心直觉
- **未剪接 RNA (Unspliced)**：刚转录出来的原始信号。
- **已剪接 RNA (Spliced)**：成熟的信号。
- **逻辑**：如果一个基因的未剪接量远高于稳态比例，说明它正在“加速”表达。
- **结果**：通过对数千个基因的速率进行矢量求和，我们可以得到每个细胞在 UMAP 空间中的“速度矢量”，直接指出它接下来将演化成哪种状态。

## 4. 算法权衡与误区

| 特性 | 轨迹推断 (Pseudotime) | RNA Velocity |
| :--- | :--- | :--- |
| **基础** | 细胞间的静态距离 | 剪接与降解的动力学 |
| **方向** | 需要手动指定起点 | 自动推断方向 |
| **敏感度** | 对大尺度过程敏感 | 对瞬时动态敏感 |

**常见误区**：
- **UMAP 形状就是轨迹**：错误。轨迹应在高维或 PCA 空间计算，UMAP 只是为了方便人类观看。
- **伪时间等于真实时间**：错误。细胞在不同阶段的停留时间不同，伪时间反映的是状态变化量。
