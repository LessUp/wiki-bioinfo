## 问题背景

在变异检测中，区分变异的生物学来源（biological origin）对于选择正确的分析方法、解释结果以及临床报告都至关重要。根据变异发生的细胞谱系和传递方式，人类变异可分为两大类：

| 特征 | Germline Variants（胚系变异） | Somatic Variants（体细胞变异） |
|-----|---------------------------|---------------------------|
| **发生时间** | 受精前（父母生殖细胞）或早期胚胎发育 | 个体发育后期或成年后的体细胞分裂 |
| **细胞分布** | 个体几乎所有细胞 | 仅存在于特定组织或细胞克隆 |
| **遗传给后代** | 可以 | 不可以 |
| **研究应用** | 遗传病诊断、种群遗传学、携带者筛查 | 肿瘤研究、克隆进化、精准医疗 |
| **检测设计** | 单样本或家系 trio 设计 | 肿瘤-正常配对（tumor-normal pair） |
| **统计模型** | 基于孟德尔遗传规律的基因型推断 | 基于克隆频率和肿瘤纯度的 VAF 分析 |

## 定义与概念

### Germline Variants（胚系变异）

**定义**：存在于生殖细胞系（germline）中的 DNA 序列变异，在个体的大部分体细胞中共有，可通过有性生殖传递给后代。

**生物学特征**：
- 在个体内通常为**合子型（zygosity）**：0/0（野生型）、0/1（杂合）、1/1（纯合）
- 遵循**孟德尔遗传规律**，可用于家系连锁分析
- 与**种群频率**密切相关，稀有胚系变异更可能具有致病性

**典型应用场景**：
- 遗传性疾病诊断（如囊性纤维化、镰状细胞贫血）
- 药物基因组学（如 CYP 基因多态性）
- 种群遗传学研究与进化分析

### Somatic Variants（体细胞变异）

**定义**：在个体发育过程中，由体细胞 DNA 复制错误或环境诱变因素导致的变异，仅存在于产生该变异的细胞及其后代克隆中。

**生物学特征**：
- 等位基因频率（VAF）反映**克隆比例**和**肿瘤纯度**：$VAF \approx \frac{f_{\text{clone}} \times \text{copy\_number}}{2 \times (1 - \text{purity}) + \text{purity} \times \text{total\_copy\_number}}$
- 可能呈现**亚克隆结构**（subclonal architecture），反映肿瘤异质性
- 同一肿瘤内可能存在多个竞争克隆，各自携带不同的突变谱

**典型应用场景**：
- 肿瘤驱动基因（driver mutations）识别
- 克隆进化追踪与转移分析
- 治疗响应监测与耐药机制研究

## 在数据与设计上的典型差异

### Germline Calling 的研究设计

**实验设计**：
- **单样本设计**：单个体的全基因组或外显子测序，适用于初步筛查；
- **家系设计（Trio）**：先证者 + 父母，用于区分新生突变（de novo）与遗传变异；
- **大家系设计**：多代家系，用于连锁分析（linkage analysis）和单倍型推断。

**核心分析任务**：
1. **基因型推断**：对于二倍体位点，判断基因型为 0/0、0/1 或 1/1；
2. **种群频率对照**：与 gnomAD、ExAC 等数据库比对，过滤常见多态性；
3. **遗传模式验证**：检查变异是否符合常染色体显性/隐性、X 连锁等遗传模式；
4. **功能注释**：预测变异对蛋白质功能的影响（如 SIFT、PolyPhen 评分）。

**统计假设**：
- 变异在样本中以稳定的合子型形式存在；
- 等位基因频率服从孟德尔遗传规律（杂合位点期望 AF ≈ 0.5）；
- 家系内一致性是真实变异的重要证据。

### Somatic Calling 的研究设计

**实验设计**：
- **肿瘤-正常配对（Tumor-Normal Pair）**：配对采集肿瘤组织和相邻正常组织，通过比对去除 germline 背景；
- **仅肿瘤设计（Tumor-Only）**：无配对正常样本时，需依赖种群数据库过滤 germline 变异；
- **多区域/多点设计**：同一肿瘤的不同区域或多时间点采样，用于研究克隆进化；
- **单细胞设计**：单细胞测序解析肿瘤异质性，直接观测克隆结构。

**核心分析任务**：
1. **体细胞变异检测**：识别肿瘤中存在、正常中不存在的变异；
2. **肿瘤纯度估计**：估计肿瘤样本中肿瘤细胞的比例；
3. **克隆频率推断**：基于 VAF 分布推断克隆组成和进化树；
4. **拷贝数变异整合**：联合分析 SNV/indel 与 CNV，校正 VAF 估计。

**统计假设**：
- 变异频率受克隆组成、拷贝数变化、肿瘤纯度共同影响；
- VAF 不局限于 0.5 或 1.0，可在 (0, 1) 范围内取值；
- 需要 panel of normals（PON）估计系统噪声水平。

## 变异检测算法（Caller）的差异

### Germline Caller

**算法特点**：
- **基于基因型似然（Genotype Likelihood）的模型**：计算 $P(\text{数据} | G)$，其中 $G \in \{0/0, 0/1, 1/1\}$；
- **种群遗传学假设**：利用 Hardy-Weinberg 平衡估算等位基因频率先验；
- **家系一致性检验**：通过家系内传递模式验证变异真实性。

**代表性工具**：
- GATK HaplotypeCaller / GenotypeGVCFs
- bcftools mpileup / call
- DeepVariant（基于深度学习的 caller）

**关键质量指标**：
- **GQ（Genotype Quality）**：基因型判定的质量分数
- **PL（Phred-scaled Likelihood）**：各基因型的似然值
- **Inbreeding Coefficient**：检测种群分层或近交信号

### Somatic Caller

**算法特点**：
- **配对样本比较模型**：同时建模肿瘤和正常样本的等位基因频率；
- **低 VAF 敏感性**：通过统计检验检测低至 1-5% 的变异等位基因频率；
- **背景噪声建模**：使用 Panel of Normals（PON）估计测序平台特定的系统误差。

**代表性工具**：
- Mutect2（GATK， tumor-normal 配对）
- Strelka2（somatic SNV/indel 检测）
- VarScan2（基于启发式规则的 caller）

**关键质量指标**：
- **TLOD / NLOD**：肿瘤/正常样本的对数似然比
- **ECNT（Event Count）**：该位点附近的事件数量
- **POPAF**：该变异在种群数据库中的等位基因频率

## 临床报告与解释的考量

### Germline 变异报告

**ACMG 分类标准**（美国医学遗传学与基因组学学会）：
- **致病性（Pathogenic）**：已有充分证据表明与疾病相关；
- **可能致病（Likely Pathogenic）**：现有证据强烈提示致病性；
- **意义不明（VUS, Variants of Uncertain Significance）**：证据不足以判断；
- **可能良性（Likely Benign）**：现有证据提示无致病性；
- **良性（Benign）**：明确无致病性。

**关键评估维度**：
- **文献证据**：ClinVar、OMIM 等数据库中的功能注释；
- **种群频率**：gnomAD 中罕见变异更可能具有致病性；
- **家系共分离**：是否符合孟德尔遗传模式；
- **功能预测**：SIFT、PolyPhen、CADD 等算法的预测评分。

### Somatic 变异报告

**体细胞变异的临床意义分层**：
- **I 级（强临床意义）**：FDA 批准的靶向治疗药物对应的变异；
- **II 级（潜在临床意义）**：指南或临床试验中提及的变异；
- **III 级（临床意义不明）**：证据不足以指导临床决策；
- **IV 级（可能良性）**：常见多态性或明确无功能影响的变异。

**关键评估维度**：
- **驱动基因状态**：是否为 COSMIC 中已知的驱动突变（driver mutation）；
- **克隆频率**：VAF 反映的克隆比例和可能的肿瘤纯度；
- **可靶向性**：是否有 FDA 批准的靶向药物或临床试验机会；
- **耐药相关**：与治疗耐药相关的突变（如 EGFR T790M）。

## 注意事项

- 不要把 germline caller 结果直接当作 somatic 解释，反之亦然；
- 使用肿瘤–正常设计时，要明确哪一套 caller/过滤规则针对 somatic；
- 在下游解释和可视化时，最好区分 germline 与 somatic 变异集合。
